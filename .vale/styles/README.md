# Vale Styles

This directory contains Vale style packages. Vale will automatically sync styles from
the StylesPath defined in `.vale.ini`.

To sync styles, run:

```bash
vale sync
```

The styles used in this project are:

- **Vale**: The default Vale style (spelling, grammar, etc.)
- **write-good**: Naive linter for English prose (avoiding weasel words, passive
  voice, etc.)
- **Google**: Google Developer Documentation Style Guide
- **Microsoft**: Microsoft Writing Style Guide

For more information on Vale styles, see [Vale Styles
Documentation](https://vale.sh/docs/topics/styles/).
